# APP ESCOM

Es un proyecto desarrollado para documentar el Análisis de Requerimientos de proyectos pequeños utilizando LaTeX. Está orientado principalmente a trabajos académicos y técnicos, y fue creado en el contexto de la Escuela Superior de Cómputo (ESCOM) del Instituto Politécnico Nacional, aunque puede ser utilizado por cualquier persona interesada en elaborar documentación formal de requerimientos.

El repositorio incluye varios archivos .tex que conforman un ejemplo completo del documento. Cada archivo representa una parte del análisis y muestra cómo se estructura y organiza la información dentro del proyecto. Estos archivos funcionan en conjunto para formar un documento final ordenado y coherente.

Para generar el documento principal se debe compilar el archivo ejemplo.tex, que integra todas las secciones y produce el PDF final con el contenido completo.
